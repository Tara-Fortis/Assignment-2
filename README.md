### Project Contents

This project showcases a website styled after the book "Tress of the Emerald Sea" written by Brandon Sanderson. The layout of the website is powere by [Bootstrap](https://getbootstrap.com/) and my custom CSS.

**Features:**
- A photo of the book cover (Tress of The Emerald Sea)
- A photo of the author (Brandon Sanderson)
- A synopsis of the book
- A hidden "buy now" form
- Navigation bar
- Carousel
- Accordion
- Tab Panel
- Google Fonts

#### Bonus features

- Blinking "buy now" button
- Collapsable payment form
- Multiple frames at once Carousel
